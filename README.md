# Система управления продуктовым складом

Веб-приложение для автоматизации складского учета на предприятии, работающем с продуктами питания.

## Функциональность

### Выбор роли
При входе в систему пользователь выбирает роль без регистрации:
- Кладовщик
- Менеджер

### Кладовщик
- Оформление приходных накладных
- Оформление расходных накладных с контролем остатков
- Просмотр текущих остатков товаров
- Просмотр истории накладных

### Менеджер
- Ведение справочников товаров (добавление, редактирование)
- Управление поставщиками (добавление, редактирование)
- Управление клиентами (добавление, редактирование)
- Формирование отчетов о движении товаров
- Контроль всех операций
- Просмотр остатков на складе

## Особенности

- Автоматический пересчет остатков при проведении накладных
- Контроль достаточности товара при отгрузке
- Проверка уникальности номеров накладных
- Информативные сообщения об ошибках
- Возможность скрытия/показа строк накладных
- Минималистичный интерфейс

## Технологии

- Go 1.24
- PostgreSQL 17
- HTMX
- Docker & Docker Compose

## Запуск приложения

### Через Docker Compose

```bash
docker-compose up -d --build
```

Приложение будет доступно по адресу: http://localhost:8443

pgAdmin будет доступен по адресу: http://localhost:5050

### Остановка

```bash
docker-compose down
```

### Просмотр логов

```bash
docker-compose logs -f app
```

## Структура базы данных

- products - товары
- suppliers - поставщики
- clients - клиенты
- incoming_invoices - приходные накладные
- outgoing_invoices - расходные накладные
- incoming_lines - строки приходных накладных
- outgoing_lines - строки расходных накладных
- stock - остатки товаров на складе

## Доступ к базе данных

```bash
docker exec -it electroshop-db psql -U appuser -d electroshop
```

## Переменные окружения

- DATABASE_URL - строка подключения к PostgreSQL
